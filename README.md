# LottoProject

* 동행복권의 openApi를 통해 로또번호를 받아서 보여주는앱
* 개인 로또번호를 생성 할 수 있음.
* flow, Hilt, Coroutine, 커스텀레이아웃등 최신 기능 개발 공부를 위해 제작중인 앱


## 기타 정보
* Platform : Android
* Language : Kotlin
* DataBase : Room(사용예정)
* Library : retrofit, DataBinding, zxing(QRcode), coroutine
* DI : Hilt
* Pattern : MVVM, Repository
